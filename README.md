## 🖼 Как заменить изображения

### Портреты

- **`images/portrait.jpg`** — вертикальный портрет для hero-секции.
  Рекомендуется соотношение `4:5`, минимум **900×1125 px**.
- **`images/about.jpg`** — фото мастера за работой для секции «Профиль».
  Рекомендуется соотношение `4:5`, минимум **800×1000 px**.

Просто положите свои фото с такими же именами в папку `images/` — сайт их автоматически подхватит.

### Работы в слайдшоу

Сейчас в слайдшоу **15 фото**: `work-01.jpg` … `work-15.jpg`.
Замените их своими с теми же именами — ничего трогать в коде не нужно.

#### Добавить ещё фото в слайдшоу

1. Положите новое фото в `images/` (например `work-16.jpg`).
2. Откройте `index.html`, найдите секцию `<!-- GALLERY · SLIDESHOW -->`.
3. Добавьте ещё один `<figure>`:

   ```html
   <figure class="slide"><img src="images/work-16.jpg" alt="Работа 16" loading="lazy" /></figure>
   ```

   Счётчик `01 / 15` и точки-индикаторы обновятся автоматически.

#### Удалить фото из слайдшоу

Удалите нужный `<figure class="slide">…</figure>` в `index.html` — счётчик снова пересчитается сам.

### Советы по фото

- Формат `.jpg` (лучше всего) или `.webp`.
- Оптимизируйте фото онлайн ([tinypng.com](https://tinypng.com)) — сайт грузится быстрее.
- Для слайдшоу лучше использовать фото примерно одинакового соотношения сторон — меньше «прыжков» при переключении.

## 🎞 Как работает слайдшоу

- ✨ **Автопрокрутка** — каждые 5 секунд
- ⏸ **Пауза при наведении мыши** (на мобильных — автоматически во время свайпа)
- 👆 **Свайп пальцем** — влево/вправо на мобильных
- ⌨️ **Стрелки ←/→** на клавиатуре
- 🔘 **Индикаторы снизу** — нажатие переключает на конкретное фото
- 🔢 **Счётчик** `01 / 15` — показывает текущее фото
- ♿ Уважает системную настройку «уменьшить анимации» — автоматически отключает автопрокрутку

Чтобы **изменить скорость автопрокрутки**, откройте `script.js` и найдите строку:

```js
const AUTOPLAY_MS = 5000;  // 5000 = 5 секунд
```

## 🚀 Как задеплоить на GitHub Pages

### Способ 1 — через веб-интерфейс GitHub (проще)

1. Зайдите на [github.com](https://github.com) и создайте новый репозиторий, например `nails-portfolio`.
2. Нажмите **"uploading an existing file"** и загрузите все файлы (`index.html`, `style.css`, `script.js`, папку `images/`).
3. Откройте вкладку **Settings → Pages**.
4. В разделе **Source** выберите ветку `main` и папку `/ (root)`, нажмите **Save**.
5. Подождите 1–2 минуты. Сайт будет доступен по адресу:

   ```
   https://ВАШ_НИК.github.io/nails-portfolio/
   ```

### Способ 2 — через командную строку

```bash
cd svetlana-portfolio
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/nails-portfolio.git
git push -u origin main
```

Затем включите Pages в Settings → Pages.

### 🌐 Свой домен (опционально)

1. В Settings → Pages → Custom domain впишите свой домен.
2. У регистратора домена создайте A-записи:

   ```
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153
   ```

3. Поставьте галочку **Enforce HTTPS**.

## ✏️ Как изменить текст

Весь контент находится прямо в `index.html`. Откройте файл в любом редакторе (например [VS Code](https://code.visualstudio.com)) и правьте текст между тегами.

Типичные места для правок:

- **Имя / заголовок** — секция `<!-- HERO -->`.
- **Телефон** — ищите `+374 55 29 30 26` (встречается 2 раза).
- **Instagram** — ищите `svet_nails__` (встречается 3 раза).
- **Опыт / годы** — секция `<!-- EXPERIENCE -->`.
- **Статистика** (50+, 70%, 50%) — в секции `<!-- PROFILE / ABOUT -->`, классы `.stat__num`.

## 🎨 Как поменять цвета

Все цвета находятся в самом верху `style.css` в блоке `:root`:

```css
:root {
  --bg:   #000000;   /* фон */
  --fg:   #ffffff;   /* основной текст */
  --gold: #c9a96e;   /* акцентный золотой */
  ...
}
```

Поменяйте hex-значения — все цвета на сайте обновятся.

## 🔠 Как поменять шрифты

В `index.html` в `<head>` подключены шрифты **Playfair Display** + **Jost** через Google Fonts. Если хотите другие шрифты — замените ссылку на Google Fonts и обновите переменные в `style.css`:

```css
--ff-serif: 'Playfair Display', serif;
--ff-sans:  'Jost', sans-serif;
```
